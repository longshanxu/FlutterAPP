# moch_flutter_app

梯控业主端

## Getting Started

1.Get Packages
2.Flutter run

图像资源文件夹assets
开发目录lib:
1. /adapter  ##适配器文件夹，对接微信，QQ等外部APP，集成登录等
2. /api      ##网络请求，http接口
3. /db       ##APP本地数据库
4. /model    ##实体类
6. /route    ##路由
7. /utils    ##实用工具，编码，解码等
8. /widget   ##小部件
9. /screens  ##页面
10. /data    ##本地存储
